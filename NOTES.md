new создаёт новый объект на основе функции-конструктора/фабрики или класса и связывает этот новый объект с прототипом.

По шагам:
1. Создаётся новый объект.
2. Новый объект получает ссылку на прототип функции-конструктора. 
3. Меняется контекст this. Функция-конструктор вызывается с контекстом this, указывающим на этот новый созданный объект.
4. Возвращается объект. Если конструктор явно не возвращает другой объект, автоматически возвращается этот созданный this.
